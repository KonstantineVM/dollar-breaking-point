readme_shl2015.txt

This file is part of the zip file shl_2015.zip

SHL_2015.zip contains a csv and txt file for the data shown in each appendix table of the
Report on Foreign Portfolio Holdings of U.S. Securities as of June 30, 2015.

The zip file is available for download from the Treasury International Capital (TIC) website: 
http://www.treasury.gov/resource-center/data-chart-center/tic/Pages/index.aspx

The report and a direct link to the zip file are available in Part A of this page:
http://www.treasury.gov/resource-center/data-chart-center/tic/Pages/fpis.aspx
