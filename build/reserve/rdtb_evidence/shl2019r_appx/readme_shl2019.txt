readme_shl2019.txt

This file is part of the zip file shl_appendix_2019.zip

SHL_appendix_2019.zip contains data for the tables in Appendix A of the 
Report on Foreign Portfolio Holdings of U.S. Securities as of June 30, 2020.

The files include:
--A csv and txt file for the data shown in each of the tables in Appendix A.

The zip file is available for download from the Treasury International Capital (TIC) website:
https://www.treasury.gov/resource-center/data-chart-center/tic/Pages/index.aspx

The report and a direct link to the zip file are available in Part A of this page:
https://www.treasury.gov/resource-center/data-chart-center/tic/Pages/fpis.aspx
