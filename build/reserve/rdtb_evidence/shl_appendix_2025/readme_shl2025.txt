This file is part of the zip file shl_appendix_2025.zip

SHL_appendix_2025.zip contains data for the tables in Appendix A of the
Report on Foreign Portfolio Holdings of U.S. Securities as of June 28, 2025.

The 25 files in the zip file are:
--A csv and txt file for the data shown in each of the tables in Appendix A.
--This readme file

The zip file is available for download from the Treasury International Capital (TIC) website:
https://home.treasury.gov/data/treasury-international-capital-tic-system

The report and a direct link to the zip file are available in Part A of this page:
https://home.treasury.gov/data/treasury-international-capital-tic-system/us-liabilities-to-foreigners-from-holdings-of-us-securities
